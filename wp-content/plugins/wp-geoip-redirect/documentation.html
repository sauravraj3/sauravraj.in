A) Installing the plugin - top

Installing the plugin is very easy just like any other wordpress plugin.
Connect to your hosting and upload the extracted zip file folder of the plugin into wp-content/plugins/
Then go to wp-admin->plugins->and activate WP GeoIP Country Redirect

B) Adding Redirect Rules - top

Adding redirect rules is very simple. Just go to WP GeoIP page in wp-admin panel and choose the target country and page/post/category.

C) Log page - top

The log page shows you the latest users redirected based on the rules you've created.
Also, if your install failed due the your hosting mysql or any other error go to the log page and please send me the exact error in there.

D) Upgrading from 2.8 to 2.9 - top

Upgrading requires you to overwrite all the plugin files by completely reinstalling the plugin (after deactivating and removing all plugin files). This will cause all existent rules to be lost
